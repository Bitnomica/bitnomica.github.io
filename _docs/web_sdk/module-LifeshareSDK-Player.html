Class: Player

LifeshareSDK~Player(containerId, src, width, height, mode, autoplay)

Player Class

Constructor

new Player(containerId, src, width, height, mode, autoplay)

source can be one of:

  • string: the player loads interprets the string as a Story.id, and plays the respective story.
  • object: { 'story': <id> }, the same
  • object: { 'channel': { 'slugs': [<SLUGS>, ...], 'type': <TYPE>, 'offset': <OFFSET> }, the player loads the latest story from the specified channel into the iframe. Read more below.
  • object: { 'url': <url> }, the player loads the story specified with url into the iframe. Url which should point to a (dynamic) story.

Loading from a channel

  var item = {
    'channel': {
      'slugs': [<SLUGS>, ...],
      'type': <TYPE>,
      'offset': <OFFSET>
     }
  }
  var player = new LifeshareSDK.Player("player", item, "800px", "450px", "default", false);
  • <SLUGS> is an array of strings containing the path of slugs pointing to the channel.
  • <TYPE> is the method of loading stories from a channel. Currently supported are: "direct" and "sub".
  • <OFFSET> is the number in the list of stories that is selected. Default is 0. With <OFFSET> = 1, the player loads the one-but-newest story, etc Read more below.

Player supports several advanced methods for loading stories from a Channel. Currently supported are:

  • Case 1. direct: Loading the latest story from a specific channel (only stories that are directly contained in the channel, but not in the subchannels)
  • Case 2. sub: Loading the latest story from a specific channel OR any of the subchanne

Example Case 1:

  var item = {
     'channel': {
       'slugs': ["mainchannel"],
       'type': 'direct',
       'offset': 0
      }
  }
  new LifeshareSDK.Player("player", item, "800px", "450px", "default", false);

Loads the first story from the list of all stories directly under "mainchannel".

Example Case 2:

   var item = {
     'channel': {
       'slugs': ["mainchannel"],
       'type': 'sub',
       'offset': 1
      }
   }
   new LifeshareSDK.Player("player", item, "800px", "450px", "default", false);

Loads the second story from the list of all stories under "mainchannel" or any of its subchannels.

Parameters:
Name Type Description
containerId string

The id of a container element in the DOM
src object

The source you want to play
width string

the preferred CSS width of the player. Use null for autofit
height string

the preferred CSS height of the player. Use null for autofit
mode string

one of: "default", "noninteractive", "modal"
autoplay boolean

start playing automatically. noninteractive also means autoplay is on. Default is false
Example
<script type="application/javascript" src="https://app.vidicrowd.com/scripts/lifesharesdk-v3.js">

<div id="player"></div>
<script type="application/javascript">
  var player = new LifeshareSDK.Player("player", "7", "800px", "450px", "default", false);
  player.addEventListener("player_ended", (e) => {
    console.log("Player Ended!");
  });
</script>
