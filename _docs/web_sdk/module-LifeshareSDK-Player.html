Class: Player

LifeshareSDK~Player(containerId, src, width, height, mode, autoplay)

Player Class

Constructor

new Player(containerId, src, width, height, mode, autoplay)

source can be one of:

  • string: the player loads interprets the string as a Story.id, and plays the respective story.
  • object: { 'story': <id> }, the player loads the story into the iframe.
  • object: { 'channel': { 'slugs': [<SLUGS>, ...], 'type': <TYPE>, 'offset': <OFFSET> }, the player loads the latest story from the specified channel into the iframe. Read more below.
  • object: { 'url': <url> }, the player loads the story specified with urlinto the iframe. Url which should point to a (dynamic) story.

Loading from a channel

Player supports several advanced methods for loading stories from a Channel. Currently supported are:

  • Case 1. direct: Loading the latest story from a specific channel (only stories that are directly contained in the channel, but not in the subchannels)
  • Case 2. sub: Loading the latest story from a specific channel OR any of the subchannels.

Also supported is to use an offset. With an offset=1, the player loads the one-but-newest story, etc...

Case 1:

  {
     'channel': {
       'slugs': [<SLUGS>, ...],
       'type': 'direct',
       'offset': <OFFSET=0>
      }
  }

Case 2:

   {
     'channel': {
       'slugs': [<SLUGS>, ...],
       'type': 'sub',
       'offset': <OFFSET=0>
      }
   }
Parameters:
Name Type Description
containerId string

The id of a container element in the DOM
src object

The source you want to play
width string

the preferred CSS width of the player. Use null for autofit
height string

the preferred CSS height of the player. Use null for autofit
mode string

one of: "default", "noninteractive", "modal"
autoplay boolean

start playing automatically. noninteractive also means autoplay is on. Default is false
Example
<script type="application/javascript" src="https://app.vidicrowd.com/scripts/lifesharesdk-v3.js">

<div id="player"></div>
<script type="application/javascript">
  var player = LifeshareSDK.Player("player", "7", "800px", "450px", "default", false);
  player.addEventListener("player_ended", (e) => {
    console.log("Player Ended!");
  });
</script>
